
# **DemoQA UI Automation using Selenium (Python)**

## **📌 Project Overview**
This project demonstrates automation of multiple UI components on the DemoQA website using **Selenium WebDriver in Python**.

The entire automation is implemented in a single script (`main.py`), where different modules are handled sequentially. The goal of this project is to showcase interaction with various web elements such as input fields, radio buttons, buttons, and JavaScript alerts.

---

## **🚀 Modules Automated**

### **1. Text Box**
- Enter user details (Name, Email, Address)
- Submit the form
- Capture and print the output

---

### **2. Radio Button**
- Select radio button options
- Validate and print the selected response

---

### **3. Buttons**
- Perform different mouse actions:
  - Double Click
  - Right Click
  - Single Click
- Capture and print corresponding messages

---

### **4. Alerts Handling**
- Handle different types of alerts:
  - **Simple Alert** → Accept
  - **Confirmation Alert** → Accept / Dismiss
  - **Prompt Alert** → Dismiss
- Capture and print alert messages

---

## **🛠️ Technologies Used**
- **Python**
- **Selenium WebDriver**
